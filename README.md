# Retail-Sales-SQL-POWER-BI
This analysis was conducted using SQL for data exploration and Power BI for interactive dashboard development
## PROJECT OVERVIEW 
Worked on a retail sales dataset to explore customer purchasing patterns, sales performance and business trends. This project involved using SQL and Power BI to answer business questions, uncover insights and build an interactive dashboard to visualize the findings and support data driven-decision-making.
## OBJECTIVES
1. To analyze overall sales performance
2. To identify top-performing product categories
3. To examine sales trends over time
4. To evaluate payment method preferences
5. To compare sales across regions
6. To understand customer age group purchasing patterns
7. To assess sales channel performance
## TOOLS USED
* SQL - Data exploration and analysis
* Power BI - Dashboard development and visualization
* Excel - Initial data review
## KEY METRICS
* Total Revenue: $45.36M
* Total Transactions: 120,000
* Total Quantity Sold: 199,549
* Average Order Value: $378
* Total Customers: 24,784
## KEY INSIGTHS
* Beauty products generated the highest revenue among all product categories.
* Customers aged 35-44 contributed the highest sales revenue.
* Revenue remained relatively balanced across all regions.
* Online and Mobile App channels slightly outperformed In-Store purchases.
* Credit Card transactions recorded the highest usage among payment methods.
* Sales showed steady fluctuations over time, indicating consistent customer demand.
